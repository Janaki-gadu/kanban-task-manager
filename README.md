# 🗂️ Kanban Task Manager

A simple and interactive Kanban Board built using HTML, CSS, and JavaScript.  
It allows users to manage tasks efficiently using drag-and-drop functionality.

---

## 🚀 Features
- ➕ Add new tasks
- 📝 Drag and drop tasks between columns
- ❌ Delete tasks
- 🎨 Colorful sticky notes UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
